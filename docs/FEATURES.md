# Nexus - Features

## Task Management

### **To-Do List**
- Create, edit, and delete tasks
- Task titles and descriptions
- Priority levels (Low/Medium/High) with color coding
- Due dates with calendar picker
- Filter by status (To Do, In Progress, Review, Done)
- Filter by priority
- Mark tasks as complete
- localStorage persistence

### **Kanban Board**
- 4 columns: To Do, In Progress, Review, Done
- Drag-and-drop tasks between columns
- Within-column reordering with visual placeholder
- Task cards show title, description, priority, and due date
- Delete tasks directly from cards

### **Dashboard**
- Task statistics overview (Total, In Progress, High Priority, Completion %)
- Due today and overdue task alerts
- Compact single-card design

## Productivity Tools

### **Pomodoro Timer**
- 25-minute focus sessions
- 5-minute short breaks
- 15-minute long breaks (every 4th session)
- Start/Pause/Reset/Skip controls
- Visual session progress indicators
- Session statistics (today, current cycle, total focus time)
- Browser notifications on session completion
- State persistence across tab switches
- Mini timer widget in sidebar when active

### **Notes**
- Create, edit, and delete notes
- Title and content fields
- Grid card layout
- Click to view full note
- Edit dialog with full text editor
- Last updated timestamps
- localStorage persistence

## System

### **Settings**
- Export all data to JSON backup file
- Import data from backup
- Storage usage statistics
- Clear individual data types (Tasks, Notes, Pomodoro)
- Clear all data with confirmation
- About section with version info

### **Navigation**
- Sidebar with all pages
- Active timer display in sidebar
- Cyberpunk-themed UI throughout
